## Hi there 👋

This organization explores explainable AI methods and tools for ECG data.

The work focuses on interpretable machine learning, clinically meaningful representations, and transparent approaches to cardiac signal analysis. The goal is to support research on models that are not only accurate, but also understandable, trustworthy, and useful in practice.

### What we work on

- Explainable models for ECG analysis
- Concept bottleneck models (CBMs)
- Unsupervised self-organizing maps (SOMs)
- Post-hoc explainability methods for deep learning time-series models
- Attribution methods and counterfactual explanations
- Interpretation, visualization, and interactive applications
- Synthetic generators for trustworthy ECG data
- Reproducible research workflows for cardiac AI
